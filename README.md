# 🌿 Growing Code - Python Fundamentals Through Garden Data

Project focused on **Python fundamentals**, exploring core concepts of the language through practical exercises based on a community garden system.

---

## 📚 About the Project

In this project, we build simple functions that cover the main foundational concepts of Python, such as:

- Input and output handling
- Variables and primitive types
- Basic arithmetic operations
- Conditional statements
- Loops (iterative and recursive)
- Functions
- Introduction to type hints

Each exercise simulates a real-world scenario within a gardening context.

---

## 🗂️ Structure

```

.
├── ex0/  # Hello Garden
├── ex1/  # Garden name
├── ex2/  # Plot area calculation
├── ex3/  # Harvest total
├── ex4/  # Plant age check
├── ex5/  # Water reminder
├── ex6/  # Count to harvest (iterative & recursive)
├── ex7/  # Seed inventory with type hints

````

---

## ▶️ How to Run

This project uses a helper file `main.py` to test the functions.

```bash
python3 main.py
````

⚠️ Important:

* Do NOT use `if __name__ == "__main__":`
* Do NOT call functions directly inside the files
* Each file must contain **only the required function**
